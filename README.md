# Joe's dwm (the dynamic window manager) laptop build

The original [dwm](https://dwm.suckless.org/) desktop manager build use to use on my ThinkPad.

Based on version 6.2.



## Dependencies

  `gcc`  
  `make`  
  `xorg`  
  `libX11`  
  `libXft`
